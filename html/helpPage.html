Design Sample

Create Design

  • Drag and drop "Layout Container" or "Node" from the left palette to the design board.
    • "Layout Container" cannot overlap with other containers or nodes upon drop. If overlaps,
      it will either automatically apply shift to avoid any overlap, or fails to add container in some cases.
  • Drag and drop a "Node" into a "Layout Container" on the design board.
    • "Layout Container" automatically does layout of its nodes inside.
    • Nodes can be moved out from its container or moved to another container.
    • "Layout Container" is just a container. Design can be done without any "Layout Container".
  • Drag "Layout Container" to move all nodes inside.
    • If the container is dropped on other containers or nodes,
      the container along with inside nodes will go back to original position.
  • Connect nodes by dragging ports of nodes.

Configure Design

  • Local configuration: click any connection. The connection will be highlighted.
    Weight and label of this connection will be shown and can be changed in the local configuration panel.
  • Global configuration: click "Global" tab in the configuration panel.
  • Drag "Zoom" on the design tool bar to zoom in/out.
  • Select "Style" on the design tool bar to change connection style.

Save Design

Click "Save" button on the right side of design tool bar.

  • If not logged in, a window will be shown asking login or signup.
  • If design is new, a window will be shown asking for design title.
  • If design is based on saved design, it will be saved with the same title but with incremented version number.

Load Design

Click "Designs" tab on the top of the page. Click "Edit Design" button to load design.

  • User needs to log in to retrieve designs.
  • If user is logged in and has unsaved changes, a window will be shown asking if save design first.

Export Design to PNG file

Click "Actions" menu on the design tool bar, and select "Export to PNG".

  • A new window/tab will be opened showing the PNG of the design.

Create New Empty Design

Click "New Design" button on the design tool bar.

  • If user is logged in and has unsaved changes, a window will be shown asking if save design first.

Login/Logout/Signup

Hover over the button on the upper right corner. A menu will be shown with login/logout/signup options.
